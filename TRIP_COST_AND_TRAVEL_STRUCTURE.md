# Trip Cost and Travel Pages Structure and Copy

This document contains the current structure and all copy from the trip costs page and travel page. Use this to make updates to costs, travel information, and content.

## Page Structure Overview

### Trip Costs Page (`/faq`)
The trip costs page consists of:
1. **Hero Section** - Page header with eyebrow, title, description, and helper text
2. **What We're Paying Now Section** - Cost cards for immediate deposits
3. **What to Plan For Section** - Cost cards for estimated expenses

### Travel Page (`/travel`)
The travel page consists of:
1. **Hero Section** - Page header with eyebrow and lead text
2. **Accordion Sections** - Expandable cards with travel information

## Component Structure

### Trip Costs Page Components

**Main Component:** `app/faq/page.tsx`

**CostCard Component:**
- Props:
  - `icon`: React.ReactNode (lucide-react icon)
  - `title`: string
  - `subtitle?`: string (optional)
  - `amount?`: string (optional)
  - `amountLabel?`: string (optional)
  - `cardBgClass?`: string (default: "bg-emerald-50")
  - `cardBorderClass?`: string (default: "border-emerald-100")
  - `iconBgClass?`: string (default: "bg-emerald-100")
  - `iconColorClass?`: string (default: "text-emerald-800")
  - `amountColorClass?`: string (default: "text-emerald-900")
  - `amountLabelColorClass?`: string (default: "text-emerald-800")
  - `children?`: React.ReactNode (body content)

**Layout:**
- Container: `max-w-xl mx-auto px-4 py-10 sm:py-14`
- Background: `bg-white`
- Sections are spaced with `mt-8`

---

### Travel Page Components

**Main Component:** `app/travel/page.tsx`

**Accordion Component:**
- Uses: `@/components/ui/Accordion`
- Props for AccordionItem:
  - `title`: string
  - `icon`: Lucide icon component
  - `preview`: string (shown when collapsed)
  - `bgColor`: string (hex color)
  - `children`: React.ReactNode (expanded content)

**Layout:**
- Container: `max-w-xl mx-auto px-4 sm:px-6 py-8 sm:py-10 pb-24`
- Background: `bg-[#FAFAFA]`

## How to Update

### Trip Costs Page

1. Edit the hero section copy in the main component
2. Update cost card data:
   - Modify amounts, labels, or descriptions
   - Adjust card styling colors if needed
   - Add or remove cost cards
3. Update section headers and descriptions

### Travel Page

1. Edit the hero section copy
2. Update accordion items:
   - Modify title, preview, or content
   - Change background colors
   - Add or remove accordion items
3. Update arrival group names and times as flight details change
